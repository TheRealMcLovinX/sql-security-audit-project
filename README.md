# SQL Security Investigation

## Overview
This project demonstrates the use of SQL to investigate potential security incidents by analyzing employee and login activity data.

## Skills Used
- SQL
- WHERE clauses
- AND, OR, NOT operators
- LIKE operator
- Date and time filtering
- Security log analysis

## Tasks Completed
- Investigated failed login attempts after business hours
- Analyzed login activity on specific dates
- Identified login attempts outside of Mexico
- Retrieved employee records for security updates
